# Korean Keyboard Layout for onboard

한글 사용자의 onboard 레이아웃 패키지

# Install

```
git clone https://github.com/hamonikr/hamonikr-onboard-layout-ko.git
cd hamonikr-onboard-layout-ko
sudo ./install
```

# Remove
```
sudo ./install remove
```

# Usage

Onboard Setting > Select Korean Layout
